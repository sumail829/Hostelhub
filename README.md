# HostelHub рџЏЁ

HostelHub is a full-stack Hostel Management System designed to simplify hostel operations such as room allocation, student enrollment, and service management. It provides a structured platform for hostel administrators to manage facilities while allowing students to easily access hostel information.

## рџљЂ Features

### Admin
- Admin authentication
- Create, update, and delete rooms
- Manage student enrollments
- Check room availability
- Manage facilities
- Manage meal plans
- Access admin dashboard

### Student
- Student registration and login
- View available rooms
- Enroll into rooms
- View hostel facilities
- View meal services

### System
- Role-based access control
- REST API architecture
- Secure authentication using JWT
- Dynamic frontend data fetching
- Responsive UI design
